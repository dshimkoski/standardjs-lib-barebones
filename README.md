# standardjs-lib-barebones

* c8 for coverage (Node.js >= 10.12.0)
* esm for module support
* microbundle for output formats
* standardjs code style
* tape for testing

## Commands

* `npm run build` - find output in dist
* `npm run example`
* `npm run watch`
* `npm test`
